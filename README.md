# Mission Résistance — app

## Description

Application mobile « Poste Radio TSF » : l'accessoire d'un grand jeu de colonie
sur le thème d'une radio militaire de la Seconde Guerre mondiale. Chaque équipe
dispose d'un poste pour émettre et réécouter des messages vocaux, appeler le QG
et progresser dans un scénario de missions à déchiffrer.

Voir la spécification produit : [`../design/SPEC.md`](../design/SPEC.md).
